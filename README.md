# PCA-LDA-Carcinoma
Reprodução de um modelo baseado em PCA-LDA para a identificação de pacientes com carcinoma através de espectroscopia RAMAN.
A biblioteca elaborada para o tratamento dos dados está em "egurian.py"

"Repeated double cross-validation applied to the PCA-LDAclassification of SERS spectra: a case study with serum samples from hepatocellular carcinoma patients"
